Signature — alag app (Softwaresolution se alag)
================================================

PC browser:
  http://signature.test/

LAN / mobile / tablet / WiFi EXTENDER (same network):
  Pehle network-url.bat chalao — wahan sahi IP dikhegi.
  Example: http://192.168.1.15:8080/
  Fixed IP (optional): http://192.168.3.50:8080/
  (port 8080 = Signature only — setup script Apache configure karta hai)

Har extender par same URL tabhi chalega jab:
  - Extender "Access Point / Repeater" mode mein ho (same WiFi name)
  - Router mode extender NA ho (alag subnet ban jati hai)
  - Server PC ka fixed IP ho (recommended 192.168.3.50)
  - Windows Firewall port 8080 allow ho

Setup (Admin — koi bhi tareeqa):

  A) Double-click:
     scripts\setup-signature-lan-network.bat

  B) PowerShell (Admin):
     cd C:\laragon\www\signature\scripts
     powershell -ExecutionPolicy Bypass -File .\setup-signature-lan-network.ps1

Fixed IP (optional):
  scripts\set-cafe-lan-ip.bat

Quick URLs check:
  Double-click network-url.bat

Softwaresolution alag rehta hai:
  http://softwaresolution.cafe/
  http://localhost/
  http://192.168.3.50/          (port 80)

Database:
  Signature        → signature_local
  Softwaresolution → softwaresolution

Files:
  Signature        → C:\laragon\www\Signature
  Softwaresolution → C:\laragon\www\Softwaresolution

Dono bilkul alag hain — ek mein change doosre mein nahi jata.
